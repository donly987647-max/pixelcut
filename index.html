자주 묻는 질문

정말 무료인가요? API 키가 필요 없나요?+
네, 완전히 무료입니다. API 키나 회원가입이 전혀 필요 없습니다. @imgly/background-removal 오픈소스 AI 모델을 브라우저에서 직접 실행합니다.
이미지가 서버에 저장되나요?+
아니요. 모든 처리는 100% 브라우저 안에서만 이루어집니다. 이미지가 외부 서버로 전송되지 않습니다.
처음 실행할 때 왜 느린가요?+
최초 실행 시 AI 모델 파일(약 50MB)을 다운로드합니다. 이후에는 브라우저에 캐시되어 빠르게 실행됩니다.
지원하는 파일 형식은?+
배경 제거는 JPG, PNG, WebP (최대 10MB), 리사이즈는 JPG, PNG, WebP를 지원합니다.
